证明力与证据能力

发表于 2013-10-10 | 分类于 杂谈 | | 阅读 次

本文大致行文思路为，首先介绍两者概念与内涵，及其纵横双向的发展脉络与各国对比，最后探讨两者内部的相互牵连关系。文章大致参照理解了他人的文章，有错误之处还望指正。

先来看证明力，证据的证明力，指证据事实对案件事实是否有证明作用和作用的程度，又称之为证据价值。

1. 证明力的发展脉络

证明力的发展脉络可以用下图来表示：

可以看到，证据的证明力经历了三个阶段。在最早期的神示证据时期，因为认知水平的落后，许多事实无从查证，为了能够解决纠纷裁判，法官只能求助与万能的神。比如有“沉水审判”，将被审判者投入水中，如果能够下沉则说明其无罪，反之则有罪。与此类似的还有中国古代神兽谛听的传说。在神示证据时期，被审判者有罪与否基本是听天由命的状态，但是法官在一定程度上也有裁量空间。比如沉水审判中，如果法官认为罪犯清白，可以往其身上绑几块石头，帮助其下沉，但是这样的自由裁量空间还是非常有限的。总体而言，与其说法官是裁判者，不如说只是仪式的主持人。

随着认知水平的逐渐发展，人们发现神示证据不太靠谱，加上中央集权下对审判权的控制，证明力进入了法定证据的阶段。在这个阶段，法律直接规定了证据链，由哪些证据可以得出何种结论。在这个阶段，虽然事实由证据认定，摆脱了不靠谱的神，但是法官在事实判断上的自由心证依旧不自由，只要有证据，便可依照法定的证据链，得出认定事实的结论。

在这里需要考虑一个问题，即证据链是否需要由法律来规定？人们逐渐发现，什么证据能证明什么内容，哪些证据可以构成证据链共同证明事实，法律无法穷尽其规定，另外，口供作为普遍的强势证据导致了刑讯逼供的滥用，同时，作为一个具有健全理智的成年人，应当具有类似的逻辑判断力，即从现有证据归纳出事实的能力，由此进入了自由心证阶段。英美法系的陪审团，就发挥着作为理智健全人判断事实的作用。但是如果将所有的证据都让陪审团进行判断，不利于陪审团工作的进行，某些证据容易产生多个争执点，混淆主要争议，偏离事实认定的重心，因此需要法官首先对陪审团能看到的证据进行预筛选。这个筛选过程需要遵循一定的原则，这些原则逐渐发展成为了证据规则。比如传闻证据规则与最佳证据规则，就是两条著名的证据规则。与之相对的是大陆法系，大陆法系即使引入了陪审团制度，也与英美法系有区别。大陆法系的陪审团与法官并非独立分开行使职能，而是相互辅助，故可以在审判过程中对证据进行认定，不需要预先的筛选。

在证明力的演变过程中，法官的自由心证不断扩大，现今在证据规则至上也能进行一定的变通。比如传闻证据规则出现了一些例外，最佳证据规则中二手证据在满足一定条件的情况下也能被采纳。

2. 证据能力的发展脉络

再来看证据能力，证据能力是指事实材料成为诉讼中的证据所必须具备的条件，及法律对事实材料成为诉讼中的证据在资格上的限制。因此，证据能力也称之为证据资格，或证据的适格性。

与证明力的发展脉络不同，证据能力直至陪审团制度的出现才跃入人们的视野。因为在证据能力是对何种证据能够作为证据进行规定，而神示证据没有这样的必要。

总体而言，证据能力就是从无到有、由紧及松发展。其中英美法系国家，由于证据规则的产生和发展与陪审团密切相关，未免陪审团因部分不法证据陷入感情与同情之偏见，需要法官对证据按照一定标准进行示明，由此发展出了一整套证据规则。但是在大陆法系国家，因为大陆法系更加重视法官的作用，法官亦是受过专业训练的人员，因此不担心法官收到非法证据的误导，从而大陆法系很少对证据能力进行限制。随着国际交流的加强，两种模式相互交融，各取所长，逐渐向中性发展，英美法系陪审团制度的使用比例逐渐缩小，而大陆法系将陪审团引入改造为类似参审模式。

3. 证明力与证据能力的关系

我国法律中证据有三性，合法性、真实性与关联性。其中真实性与关联性是我国法律特别关注的内容，代表了证据的自然基础，是证明力的概念范畴。而相对较为忽略的合法性内容，是证据的法律基础，这一部分则更贴近证据能力的范畴。与之不同的是，英美法系似乎将真实性归入证据能力的范畴。对此我存有疑惑，对这样的归类自己也不敢肯定。比如我们说，传闻证据排除，是因为传闻证据很可能是虚假的，那么这是真实性的内容，传闻证据排除规则则是证据规则，属于证据能力范畴，这在英美法系是说得通的。但是在大陆法系，我们有传闻证据证明力弱于原始证据，这是对证明力强弱的表述，是否就代表我国就将真实性纳入证明力范畴了呢？

关于证明力与证据能力，有一个争论，类似先有鸡还是先有蛋的争论。一个证据，是否以其具有证据能力为前提，也就是说，如果一个证据不符合证据能力，即不具有证据资格，自然无从谈论其证明力问题。这是我国大多数学者的通说。也就是说，通说认为证据能力是证明力的前提。但是我们说，在自由心证下对证据的审核，不是一板一眼的，而是灵活机变的。倘若因某个证据不具有证据能力，就一味不再研究其证明力与证明对象，那么是否又回到了呆板的绝对的证据规则上了，某些例外下的证据如何进行审查，另外庭审中如何发挥法官的作用，多个非法、弱势的证据是否能够推翻强力证据？特别是在民事活动中这样的探讨是否有必要？

4. 证据能力对证明力的限制

这部分内容其实已经在之前讲了蛮多，积极性限制是作为证据的前提条件，消极性限制是作为证据的否定要件。以最佳证据规则为例，其要求提供原件。这里的原件在英美法系指的是文书、记录或者照片。也如这样表述更为适当：“除非不可能，否则都应出示文书本身”。那么与案件有关的实物是否纳入最佳证据规则？至少英美法系并不包含，而在大陆法系，法官应当根据最佳证据规则背后的政策法理进行衡量，中国则是肯定了实物证据适用最佳证据规则。最佳证据规一般有几个例外：当事人无异议、原件损毁、原件在对方手中、原件在第三人手中、官方记录、其他。如广泛传播的字帖、无法移动的墙、原件损毁由举证方故意行为是例外的例外。
