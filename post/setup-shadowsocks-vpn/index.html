自建VPN搭建Shadowsocks服务器科学上网全攻略（HostUS）

发表于 2016-10-21 | 分类于 IT | | 阅读 次

一直以来，我都对公有VPN服务器的安全性存有疑问。为了更好的科学上网，经过多次亲身实战，简要写了一篇攻略。

服务器选择

因为我使用电信网络为主，经过对比HostUS、DigitalOcean、UFOVPS、Aliyun HongKong等多家云服务器供应商实测结果，线路上HostUS属于半CN2线路，DigitalOcean无CN2线路，而位于香港的UFOVPS、Aliyun HongKong虽然号称与大陆地区是CN2直连，但鉴于VPN用于科学上网，香港的国际带宽昂贵，在访问时都从美国等地绕一圈，实际效果反而不如美国的VPS。

因此经过对比，我最终选择了HostUS的LA服务器，经测试延迟200ms左右，能够满足一般的YouTube需求，价格也非常亲民，每月最低不到5美元。传送门：HostUS。

新建VPN服务器

为便于安装锐速（ServerSpeeder）加强访问效果，我们需要新建KVM的服务器。在HostUS的VPS选择界面选择一款适合自己的（一般而言最低配的即可）服务器配置，填写相应信息后付款，等待一段时间的人工审核（一般一小时左右），即可看到服务器状态变为“Active”，可以使用了。

修改VPN服务器系统

我习惯于使用Ubuntu系统，而HostUS默认新建的是CentOS系统，故需要在服务器设置左侧边栏选择Rebuild，并在右侧选择Ubuntu 14.04系统。

下方的Hostname随便取个名字，在Root Password处填入自己喜爱的密码，SSH Key可以先空着。然后点击下方的Rebuild。

远程登录VPN服务器

下载Putty软件（Windows下），填写Host Name中填写服务器的IP地址，Port中填写端口为22，，然后点击下方的Open即可开始连接。

在出现Login as:的提示后，输入root，会提示输入root密码。root密码就是在上一步Rebuild中所填写的Root Password内容。在输入密码时，是不会提示有键入信息的，输完直接按回车即可。

如果使用OS X系统或者Linux系统，可以直接打开终端，输入ssh root@服务器IP地址进行连接。
